# web_search
搜尋網路資訊，回傳搜尋結果摘要與 URL。
## 參數
- query (str): 搜尋查詢字串
- max_results (int, 選填): 最大結果數，預設 5
## 使用時機
- 需要查詢最新資訊、事實查核時使用。
- 需要找到特定資源的 URL（如資料庫頁面、下載連結）時使用。
## 重要提醒
- web_search 只回傳搜尋結果摘要，不會下載檔案。
- 找到下載 URL 後，必須用 download_file 下載檔案。
- 找到網頁 URL 後，用 web_browser 讀取完整網頁內容。
- 不要反覆搜尋同樣的東西。如果已經找到需要的 URL，就去下載或瀏覽它。
